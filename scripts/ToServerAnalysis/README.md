First, to get the processed data, you can check the `full-analysis.ipynb` file that includes the full analysis to create them. Then, to prepare your environment, you should to create the conda env aab_rtp:

> conda env create -f env.yaml

After that you can do in local the `prepareCVSKF.py` to split the different matrix in stratified folds (for cross-validation):

> python3 prepareCVSKF.py

Finally, in your server (considering that you run in SLURM with sbatch), you can run `Submit_aab_rtp.sh`
